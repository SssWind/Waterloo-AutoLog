Introduction

▷ You will be automatically logged in every time you access the supported pages.

Notes

Privacy

The extension do not track any of your information

All information are stored 100% locally in your browser.

Password and WatCard PIN are encrypted before stored. The random key used for encryption is automatically generated after installation.

For your account safety, do not use this extension on any public computers.

The original code of this extension is not encrypted, you can view it on GitHub or in your browser's extension folder (Google the tutorial).
