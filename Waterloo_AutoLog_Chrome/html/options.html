Privacy
Keep your Waterloo account safe

Waterloo AutoLog does not track any of your information.

Everything you entered here is stored 100% locally in the computer and is not synced with your browser account.

Password and WatCard PIN are encrypted before saving. The random key used for encryption was randomly generated after installation.

You can check the source code on GitHub at any time. See related links in the section below.

The extension uses "" to store your settings locally in Profile Path.

Visit to get detailed information.

The default profile on Windows 10 is located at (depends on system configuration):
C:\Users\user_name\AppData\Local\\User Data\Default.

For lab computers, C:\Users\user_name or any private storage space (e.g. N:\) can only be accessed by yourself and the IST administrators. If your profile is stored under these locations, you may use this extension. Otherwise do not.

For library computers or any other public computers, DO NOT use this extension.

After you remove this extension, all the data is automatically cleared.
